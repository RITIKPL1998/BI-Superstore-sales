# Global Superstore BI Project

## 📌 Project Summary
A full-stack BI pipeline using Apache Airflow, Python ETL, and Power BI dashboards on the Global Superstore dataset.

## 🧰 Tech Stack
- Apache Airflow
- Python (Pandas, SQLite)
- Power BI / Tableau
- Git & GitHub

## 🔁 Pipeline Flow
1. Extract: Source data from CSV
2. Transform: Clean & enrich with KPIs
3. Load: Save into SQLite database
4. Visualize: Power BI dashboards
5. Orchestrate: Scheduled via Airflow
